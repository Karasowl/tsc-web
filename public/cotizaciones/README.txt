INSTRUCCIONES PARA MIGRAR LAS COTIZACIONES
==========================================

1. Copia todos los archivos HTML de cotizaciones de tu proyecto anterior aquí.
   
2. Los archivos deben mantener sus nombres originales, por ejemplo:
   - cotizacion-fraccionamientoSanJose.html
   - cotizacion-onirik.html
   
3. Las URLs seguirán funcionando igual sin la extensión .html:
   - https://guardias.tscseguridadprivada.com.mx/cotizacion-fraccionamientoSanJose
   - https://guardias.tscseguridadprivada.com.mx/cotizacion-onirik

4. Asegúrate de copiar también cualquier recurso (CSS, imágenes, JS) que usen estas cotizaciones.

5. Los archivos CSS y JS específicos de las cotizaciones pueden ir en:
   - public/cotizaciones/css/
   - public/cotizaciones/js/
   - public/cotizaciones/img/

6. Si las cotizaciones hacen referencia a recursos compartidos del sitio principal,
   asegúrate de actualizar las rutas relativas según sea necesario.
